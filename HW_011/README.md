# Homework 11: Class Activity Submission

This assignment is to submit the work done during the class activity.

## Instructions

1.  Take **5 screenshots** of your work during the class activity.
2.  Create a file named `homework.md` in this directory.
3.  Embed your 5 screenshots in the `homework.md` file.
4.  Push your changes to GitHub.

## File Structure

Your submission should look like this:

```
HW_011/
  ├── homework.md
  ├── screenshots/      (Suggested folder for your images)
  │    ├── screen1.png
  │    ├── screen2.png
  │    ├── ...
  └── README.md         (This file)
```

## `homework.md` Example

```markdown
# My Class Activity

## Screenshot 1
![Description](screenshots/screen1.png)

## Screenshot 2
![Description](screenshots/screen2.png)

...
```
